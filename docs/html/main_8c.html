Detailed Description

Exibe a temperatura interna do RP2040 em um display, com efeitos visuais em matriz de LEDs WS2812B.

Este projeto demonstra como ler a temperatura interna do microcontrolador e exibir no display. Também inicializa uma matriz de LEDs e desenha um quadrado visual usando LEDs verdes.
