Detailed Description

Interface para controle de display OLED SSD1306 via I2C.

Fornece funções para inicialização, limpeza e exibição de texto no display OLED, utilizando a biblioteca ssd1306. Os pinos e parâmetros de comunicação I2C são definidos aqui.
